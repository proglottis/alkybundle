AlkyBundle
----------

Compiling Mini-Howto
--------------------
Prerequisites:
 * CMake (http://www.cmake.org/)
 * Qt 4 (http://www.qtsoftware.com/)
 * Boost (http://www.boost.org/)
 * SQLite3 (http://www.sqlite.org/)
 * CxxTest (Optional, http://cxxtest.com/)

Linux (Makefiles):
 1. $ mkdir build
 2. $ cd build
 3. $ cmake ..
 4. $ make
 5  $ ./bin/alkybundle
